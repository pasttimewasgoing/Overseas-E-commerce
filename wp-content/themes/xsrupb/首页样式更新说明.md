# 首页样式更新说明

## ✅ 已完成的更新

### 1. 创建了专门的首页样式文件
**文件**: `assets/css/home.css`

完全基于原型图 (`原型图/首页/index.html`) 的样式，包括：

#### 轮播图样式
- 圆角边框 (16px)
- 柔和阴影效果
- 平滑的淡入动画
- 圆形导航按钮
- 动态指示点

#### 产品卡片样式（5列布局）
- **网格布局**: 5列，间距 24px
- **卡片效果**:
  - 圆角 12px
  - 悬停上浮 8px
  - 柔和阴影
  - 图片缩放效果
- **图片区域**: 220px 高度，灰色背景 (#f5f5f5)
- **标题**: 16px，深色 (#1a1a1a)，加粗
- **描述**: 13px，灰色 (#888)
- **价格**: 22px，红色 (#ff4757)，加粗
- **按钮**: 黑色背景，白色文字，悬停动画

#### 产品标签
- **NEW**: 绿色 (#2ed573)
- **HOT**: 红色 (#ff4757)
- **SALE**: 橙色 (#ffa502)

#### 区块样式
- **标题**: 36px，居中，底部装饰线
- **间距**: 上下 80px
- **背景**: 交替使用白色和浅灰色 (#fafafa)

#### 热门推荐标签页
- 圆角按钮 (25px)
- 激活状态：黑色背景
- 悬停效果：灰色背景

#### 视频卡片
- 3列网格布局
- 圆角 16px
- 播放按钮居中
- 悬停缩放效果

## 🔧 自定义调整

### 修改列数
编辑 `assets/css/home.css`：
```css
.product-grid {
    grid-template-columns: repeat(5, 1fr); /* 改为 4 或 6 */
}
```

### 修改卡片高度
```css
.product-card .product-img {
    height: 220px; /* 改为其他值 */
}
```

### 修改价格颜色
```css
.product-price {
    color: #ff4757; /* 改为其他颜色 */
}
```

### 修改按钮样式


## 🐛 故障排除

### 样式没有生效？
1. 清除所有缓存
2. 检查文件是否存在：`assets/css/home.css`
3. 检查浏览器控制台是否有 404 错误
4. 硬刷新：Ctrl+F5

### 布局错乱？
1. 检查是否有其他插件的 CSS 冲突
2. 检查浏览器窗口宽度
3. 检查响应式断点

### 图片不显示？
1. 确保产品已设置特色图片
2. 使用 `check-product-images.php` 检查
3. 清除缓存
