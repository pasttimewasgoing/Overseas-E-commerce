专为电子产品商城设计的 WordPress 主题，深度集成 WooCommerce 电商功能。

== Description ==

XSRUPB 是一个功能完整的电商主题，专为电子产品商城设计。主题深度集成 WooCommerce，提供产品展示、分类浏览、购物车管理等核心电商功能。

主要特性：

* 完整的 WooCommerce 集成
* 响应式设计，支持多设备访问
* 产品列表和详情页面
* 购物车和结账功能
* 产品搜索和分类过滤
* AJAX 购物车操作
* 多语言支持
* 性能优化
* 安全性增强

== Installation ==

1. 上传主题文件到 `/wp-content/themes/xsrupb/` 目录
2. 在 WordPress 后台激活主题
3. 确保已安装并激活 WooCommerce 插件
4. 在"外观 > 自定义"中配置主题设置

== Frequently Asked Questions ==

= 这个主题需要什么插件？ =

必需插件：
- WooCommerce 8.0 或更高版本

推荐插件：
- Contact Form 7（联系表单）
- YITH WooCommerce Wishlist（愿望清单）

= 主题支持哪些浏览器？ =

主题支持所有现代浏览器，包括：
- Chrome（最新版本）
- Firefox（最新版本）
- Safari（最新版本）
- Edge（最新版本）
