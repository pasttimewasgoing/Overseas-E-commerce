# TechVision WordPress主题

专为电子产品商城设计的现代化WordPress主题。

## 主题特性

### 核心功能
- 响应式设计，完美适配各种设备
- 自定义产品文章类型
- 产品分类和标签系统
- 轮播图展示
- 多标签产品展示
- 视频展示区域
- WooCommerce支持
- 自定义Logo支持
- 多语言准备就绪

### 首页模块
1. **导航栏** - 包含主菜单、产品下拉菜单、语言切换、用户中心和购物车
2. **轮播图** - 支持4个轮播项，自动播放
3. **新品上线** - 展示最新产品，带NEW标签
4. **热门推荐** - 5个分类标签切换展示
5. **产品视频** - 视频展示区域
6. **更多产品** - 促销产品展示，带SALE标签
7. **页脚** - 4列小工具区域

### 自定义文章类型
- **产品 (Product)** - 用于展示电子产品
  - 支持特色图片
  - 自定义字段：价格、标签类型
  - 产品分类和标签

### 小工具区域
- 主侧边栏
- 页脚区域1-4（4列布局）

### 菜单位置
- 主导航菜单
- 页脚菜单

## 安装说明

1. 下载主题文件
2. 上传到 `wp-content/themes/techvision/` 目录
3. 在WordPress后台 外观 > 主题 中激活主题
4. 配置菜单：外观 > 菜单
5. 配置小工具：外观 > 小工具

## 推荐插件

- **WooCommerce** - 电商功能
- **Polylang** 或 **WPML** - 多语言支持（必须安装其中一个才能显示语言切换器）
- **Contact Form 7** - 联系表单
- **Yoast SEO** - SEO优化

## 多语言设置

主题支持以下多语言插件：

### 使用 Polylang
1. 安装并激活 Polylang 插件
2. 进入 语言 > 语言设置
3. 添加需要的语言（如中文、英文）
4. 语言切换器会自动显示在导航栏右侧
5. 翻译菜单、页面和产品内容

### 使用 WPML
1. 安装并激活 WPML 插件
2. 完成 WPML 设置向导
3. 添加需要的语言
4. 语言切换器会自动显示在导航栏右侧
5. 使用 WPML 翻译管理器翻译内容

注意：如果没有安装多语言插件，语言切换器不会显示。

## 自定义产品

### 添加产品
1. 进入后台 产品 > 添加新产品
2. 填写产品标题和描述
3. 设置特色图片
4. 添加自定义字段：
   - `product_price` - 产品价格
   - `product_badge` - 标签类型（new/hot/sale）
5. 选择产品分类
6. 发布产品

### 产品分类
- 黑白电子纸显示屏
- 多色电子纸显示屏
- 彩色电子纸显示屏
- 开发套件
- 触控与灯光

### 产品标签
- `new` - 新品标签（绿色）
- `hot` - 热门标签（红色）
- `sale` - 促销标签（橙色）

## 轮播图设置

1. 创建分类 "carousel"
2. 发布文章到该分类
3. 设置特色图片
4. 轮播图会自动显示最新4篇文章

## 视频设置

1. 创建分类 "videos"
2. 发布文章到该分类
3. 设置特色图片作为视频封面
4. 添加自定义字段 `video_url` 设置视频链接
