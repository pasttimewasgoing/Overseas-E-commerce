# TechVision 主题安装指南

## 快速开始

### 1. 激活主题
1. 登录WordPress后台
2. 进入 外观 > 主题
3. 找到 TechVision 主题
4. 点击"激活"

### 2. 配置菜单
1. 进入 外观 > 菜单
2. 创建新菜单，命名为"主菜单"
3. 添加页面/分类到菜单
4. 在"菜单位置"中选择"主导航菜单"
5. 保存菜单

### 3. 配置小工具
1. 进入 外观 > 小工具
2. 配置以下小工具区域：
   - 页脚区域1：添加"文本"小工具，输入公司介绍
   - 页脚区域2：添加"文本"小工具，输入联系方式
   - 页脚区域3：添加"自定义菜单"小工具
   - 页脚区域4：添加"文本"小工具，输入社交媒体链接

### 4. 设置Logo
1. 进入 外观 > 自定义 > 站点身份
2. 上传Logo图片
3. 调整Logo尺寸
4. 发布更改

### 5. 创建产品分类
1. 进入 产品 > 产品分类
2. 创建以下分类：
   - 黑白电子纸显示屏
   - 多色电子纸显示屏
   - 彩色电子纸显示屏
   - 开发套件
   - 触控与灯光
   - 其他配件

### 6. 添加产品
1. 进入 产品 > 添加新产品
2. 填写产品信息：
   - 标题：产品名称
   - 内容：产品详细描述
   - 特色图片：产品图片
3. 添加自定义字段：
   - 字段名：`product_price`，值：产品价格（如：299.00）
   - 字段名：`product_badge`，值：标签类型（new/hot/sale）
4. 选择产品分类
5. 发布产品

### 7. 设置轮播图
1. 进入 文章 > 分类目录
2. 创建分类"carousel"
3. 进入 文章 > 写文章
4. 创建4篇文章：
   - 标题：轮播图标题
   - 摘要：轮播图描述
   - 特色图片：轮播图背景（建议尺寸：1200x400）
   - 分类：选择"carousel"
5. 发布文章

### 8. 设置产品视频
1. 创建分类"videos"
2. 创建文章并分配到"videos"分类
3. 设置特色图片作为视频封面
4. 添加自定义字段：
   - 字段名：`video_url`
   - 值：视频链接（YouTube、优酷等）
5. 发布文章

### 9. 主题自定义设置
1. 进入 外观 > 自定义 > TechVision 主题设置
2. 配置以下选项：
   - 首页设置：选择显示/隐藏各个模块
   - 颜色设置：设置主色调
   - 联系信息：填写电话、邮箱、地址
   - 社交媒体：填写社交媒体链接
3. 发布更改

## 推荐插件安装

### WooCommerce（电商功能）
```
1. 进入 插件 > 安装插件
2. 搜索"WooCommerce"
3. 安装并激活
4. 按照设置向导配置商店
```

### Polylang（多语言）
```
1. 进入 插件 > 安装插件
2. 搜索"Polylang"
3. 安装并激活
4. 进入 语言 > 语言
5. 添加中文和英文
6. 语言切换器会自动显示在导航栏
7. 翻译菜单和内容
```

或者使用 WPML：
```
1. 购买并下载 WPML 插件
2. 上传并激活插件
3. 完成设置向导
4. 添加需要的语言
5. 语言切换器会自动显示
```

注意：必须安装 Polylang 或 WPML 其中一个，语言切换器才会显示。

### Contact Form 7（联系表单）
```
1. 搜索并安装"Contact Form 7"
2. 激活插件
3. 创建联系表单
4. 在页面中使用短代码
```

## 常见问题

### Q: 产品不显示怎么办？
A: 确保已经创建产品并设置了正确的自定义字段（product_price 和 product_badge）

### Q: 轮播图不显示？
A: 检查是否创建了"carousel"分类并发布了文章到该分类

### Q: 菜单下拉不工作？
A: 确保JavaScript文件正确加载，检查浏览器控制台是否有错误

### Q: 如何更改主题颜色？
A: 进入 外观 > 自定义 > TechVision 主题设置 > 颜色设置

### Q: 如何添加更多产品分类标签？
A: 编辑 template-parts/hot-products.php 文件，在 $categories 数组中添加新分类
