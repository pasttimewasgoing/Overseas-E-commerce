管理WordPress后台显示的菜单项，可以根据需要隐藏特定菜单。

== Description ==

后台菜单管理器允许你轻松控制WordPress后台显示的菜单项。

主要功能：
* 选择性隐藏后台菜单项
* 管理员始终可见所有菜单
* 简洁的设置界面
* 轻量级，不影响性能

== Installation ==

1. 上传插件文件到 `/wp-content/plugins/admin-menu-manager/` 目录
2. 在WordPress后台"插件"菜单中激活插件
3. 进入"设置" > "菜单管理"配置要隐藏的菜单

== Frequently Asked Questions ==

= 管理员能看到被隐藏的菜单吗？ =

可以。管理员角色始终可以看到所有菜单项。

= 如何恢复隐藏的菜单？ =

进入设置页面，取消勾选对应的菜单项即可。
